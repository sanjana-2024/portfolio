Quickdy Project

Quickdy is a fast-paced, dynamic web application designed to streamline online services for users. My responsibilities included: User Interface Development: I contributed to building responsive and visually appealing user interfaces using HTML, CSS, and Bootstrap. This involved creating layouts that were both user-friendly and aligned with the project's branding guidelines. Cross-Browser Compatibility: Ensuring that the application functioned seamlessly across different browsers was a key focus. I performed thorough testing and made necessary adjustments to achieve consistent performance and appearance. Collaboration and Feedback: I collaborated closely with designers and back-end developers to ensure a cohesive development process. Regular feedback sessions helped in refining the design and functionality of the application.

Dangal Arena Project

Dangal Arena is a platform dedicated to sports and entertainment, providing users with information, schedules, and updates. In this project, I primarily focused on HTML and CSS to create a visually appealing and responsive user interface. My contributions included: Static Page Layouts: I developed the main layout and structure of the website using HTML, ensuring a clean and organized codebase. This involved setting up headers, footers, navigation menus, and content sections. Styling and Design: I applied CSS to style the website, focusing on creating a cohesive and engaging visual theme. This included selecting color schemes, typography, and spacing to enhance readability and aesthetic appeal. Responsive Design: I implemented responsive design techniques using CSS media queries. This ensured that the site was accessible and user-friendly across various devices, including desktops, tablets, and smartphones.
